# Aromix Project Templates

This repository serves as the official template source for the [Aromix CLI](https://github.com/aromixjs/aromix/tree/main/packages/cli).

## Templates

All templates use [Handlebars](https://handlebarsjs.com/) for dynamic content injection.

- `aromix.build.ts.hbs`: Build configuration.
- `package.json.hbs`: Project manifest.
- `README.hbs`: Starter README for new projects.
- `tsconfig.json.hbs`: TypeScript configuration.
- `.gitignore`: Git ignore file.
- `src/index.ts.hbs`: Entry point file.

## Usage

These templates are maintained and versioned here, and fetched by the Aromix CLI during project scaffolding.
